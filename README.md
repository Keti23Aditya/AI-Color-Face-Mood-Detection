# AI Color, Face & Mood Detection System 🎯

## 📌 Overview
This project is a real-time computer vision application that detects:
- Blue objects (highlighted with bounding boxes)
- Red objects (with blinking effect)
- Human faces and mood (Happy or Serious)

It uses OpenCV and Haar Cascade classifiers to perform detection using a webcam.

---

## 🚀 Features
- 🎨 Color Detection (Blue & Red)
- 👤 Face Detection
- 😊 Mood Detection (Smile-based)
- ⚡ Real-time Processing using Webcam
- 🔴 Red object blinking visualization

---

## 🛠️ Technologies Used
- Python
- OpenCV
- NumPy
- Haar Cascade Classifier
